Welcome to your new Hugo blog with Tailwind CSS v4! This template combines the power of Hugo's static site generation with Tailwind's utility-first CSS framework.

## Features

This blog template includes:

- **Beautiful Typography**: Using the `@tailwindcss/typography` plugin for perfect prose styling
- **Dark Mode**: Automatic dark mode with manual toggle
- **Responsive Design**: Looks great on all devices
- **Fast Performance**: Static HTML with minimal CSS
- **SEO Friendly**: Proper meta tags and semantic HTML

## Getting Started

Create new posts with:

```bash
hugo new posts/my-new-post.md
```

Then edit the generated file and set `draft: false` when ready to publish.
