# Hello, World!

This is a minimal Hugo site with **Tailwind CSS v4** integration.

## Getting Started

Edit this file at `content/_index.md` to customize your homepage.

## Features

- ✅ Hugo Static Site Generator
- ✅ Tailwind CSS v4
- ✅ Dark Mode Support
- ✅ Responsive Design
- ✅ Fast & Optimized

## Next Steps

1. Create new pages: `hugo new about.md`
2. Customize styles in `assets/css/main.css`
3. Edit templates in `layouts/`
4. Build for production: `npm run build`
